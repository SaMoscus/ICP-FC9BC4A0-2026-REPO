# Environment Setup Guide

## Objective
Linux environment with the tools required for the DevOps internship projects.

---

## System Information

**Operating System:** Ubuntu 22.04 LTS

**Shell:** Bash

**Version Control:** Git

**Container Platform:** Docker

---

## Step 1: Update the System

```bash
sudo apt update
sudo apt upgrade -y
```

---

## Step 2: Install Git

```bash
sudo apt install git -y
```

Verify installation:

```bash
git --version

## Step 4: Install Docker

```bash
sudo apt install docker.io -y
```
Verify:

```bash
docker --version
